# News Explorer – Frontend

Projeto inicializado com Vite + React para o aplicativo News Explorer do bootcamp.

## Requisitos
- Node.js 18+ (recomendado Node 20). Em WSL, use WSL 2 para melhor compatibilidade.

## Scripts
- `npm install` — instala dependências.
- `npm run dev` — ambiente de desenvolvimento com Vite.
- `npm run build` — build de produção.
- `npm run preview` — serve o build localmente.
- `npm run lint` — analisa o código com ESLint.

## Estrutura
- `src/components` — componentes React.
- `src/utils` — helpers e chamadas de API.
- `src/images` — imagens estáticas.
- `src/vendor` — fontes e outros assets de terceiros.
